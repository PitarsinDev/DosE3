### To install

```
npm i -g dose3
```

## Script
```
dose -v  ----> check version
```
```
dose update  ----> update version
```

* odeJS<br/>

* ReactJS (Front-end)

* ExpressJS (Server side)
* Actix-web (server side performance)
* Tauri (Mobile & Desktop)
<br/>

## Inside
* Zustand (Global state)
* SCSS (css lib)
* Vitest (unit test tool)
* VitePWA (mobile and desktop)
* Prisma (ORM)
* Mongoose (ODM)
